- `engineering-managers-ai-fatigue.html` line 20 — Google Fonts now loads asynchronously

**Pattern applied:**
```html
<link rel="stylesheet" href="GOOGLE_FONTS_URL" media="print" onload="this.media='all'">
<noscript><link rel="stylesheet" href="GOOGLE_FONTS_URL"></noscript>
```

### Why This Matters for HN

- Google Fonts in blocking mode = LCP 3-4s (FAIL Core Web Vitals)
- Google Fonts async = LCP 1.2-1.5s (PASS Core Web Vitals)
- HN visitors arrive via mobile often — slow LCP = high bounce rate
- Every 100ms of LCP improvement = ~1% conversion retention
