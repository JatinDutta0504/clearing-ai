## Fixes Applied

### 1. JSON-LD Schema (3 blocks added to `<head>` of index-hn.html)

**Organization schema:**
```json
{ "@type": "Organization", "name": "The Clearing", "url": "https://clearing-ai.com",
  "description": "...", "foundingDate": "2026" }
```

**Article schema (E-E-A-T signal for HN content):**
```json
{ "@type": "Article", "headline": "I built a free AI fatigue recovery tool...",
  "description": "We surveyed 2,047 engineers...", "author": {...},
  "datePublished": "2026-03-01", "dateModified": "2026-04-01" }
```

**BreadcrumbList schema:**
```json
{ "@type": "BreadcrumbList", "itemListElement": [
    { "@type": "ListItem", "position": 1, "name": "Home", "item": "https://clearing-ai.com/" }
  ]}
```

All 3 blocks validated ✅ (python3 json.loads — 100% valid)

### 2. Accessibility Fixes (index-hn.html)

**Skip-to-content link** (WCAG requirement):
- Added `<a href="#main" class="skip-link">Skip to main content</a>` after `<body>`
- Added CSS: off-screen by default, visible on `:focus`
- Added `id="main"` to hero section

**FAQ ARIA upgrade:**
- Added `aria-expanded="false"` to all 6 FAQ buttons
- Upgraded `toggleFaq()` function to toggle `aria-expanded` attribute on open/close

**Email input:**
- Added `aria-label="Email address"` to newsletter form input

**HTML validation:** 0 structural errors ✅
