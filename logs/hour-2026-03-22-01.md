## What Was Built This Hour

### Interactive AI Fatigue Self-Assessment Quiz

A fully functional, vanilla JavaScript quiz embedded directly in `index.html`.

**The experience:**
- Landing card with intro text and "Take the assessment →" CTA
- 5 thoughtfully written questions about real engineer behavior patterns
- Smooth single-question flow with animated progress bar
- Answer selection highlights instantly, auto-advances on desktop after 600ms
- Back/Next navigation for deliberate users
- Results screen with: emoji+score ring, tier label, personalized paragraph, CTA link to a relevant page, per-answer breakdown with animated fill bars, retake button

**The 5 questions:**
1. "When you sit down to write code, what happens first?" (measures autopilot tendencies)
2. "How do you feel on Sunday evening before a work week?" (measures emotional depletion)
3. "When you solve a hard problem, where does the satisfaction come from?" (measures connection to craft)
4. "How often do you review AI-generated code without fully understanding it?" (measures epistemic abdication)
5. "If a junior engineer asked you to explain a system you built this year..." (measures ownership/authorship)

**The 4 scoring tiers (score 0–15):**
- 🌿 0–3: "You're holding up well" → links to why.html
- 🌤 4–7: "Some fatigue is showing" → links to tips.html
- 🌧 8–11: "Real AI fatigue is present" → links to stories.html
- 🌑 12–15: "You need a real break" → links to resources.html

Each tier has a full paragraph of warm, human-written copy — not generic platitudes.
