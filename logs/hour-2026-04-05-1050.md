### 3. Accessibility Fix — index-hn.html
**Problem:** FAQ buttons missing `aria-controls` (links questions to answers), `aria-expanded` not dynamically updated, no `role="region"` on answer divs.

**Fix applied:**
- Added `id="faq-1"` through `id="faq-6"` to each `.faq-a` answer div
- Added `aria-controls="faq-N"` to each FAQ button
- Added `aria-labelledby="faq-btn-N"` to each answer region
- Added `role="region"` to answer divs
- Added `role="list"` and `role="listitem"` to FAQ structure
- Added `aria-hidden="true"` to decorative arrow spans
- Updated `toggleFaq()` JS to dynamically set `aria-expanded="true/false"`

**Result:** 18/18 accessibility checks now PASS on index-hn.html ✅
