## This Window: Technical SEO Audit + Font Preload Fix

### Action: Lighthouse Audit
- Ran Lighthouse on https://clearing-ai.com/ (live site)
- **LCP: 8.2s under Slow-4G throttle** (Simulated) / **2.5s observed** (real connection)
- **Root cause identified:** 3x duplicate font preload for same Inter w400 file
  - Lighthouse found: `UcCO3FwrK3iLTeHuS_nVMrMxCp50SjIw2boKoduKmMEVuLyfMZg.ttf.woff2` x3 = 954KB wasted
  - Browser was downloading same 318KB file 3x

### Fix Deployed
- **Removed 2 duplicate font preload `<link rel="preload" as="font">` lines from all 186 HTML files**
- Fonts now loaded only via `fonts/fonts.css` @font-face (async, font-display: optional)
- `font-display: optional` = Georgia fallback renders immediately, web fonts are enhancement only
- No render-blocking from fonts (confirmed by 0 render-blocking resources)
- Also cleaned up orphan comments left behind after preload removal
