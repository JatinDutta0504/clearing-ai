## CLS Root Cause

Georgia-Fallback was declared with:
```css
size-adjust: 112.5%;
ascent-override: 90%;
descent-override: 24%;
```

This made Georgia render **12.5% LARGER** than Lora. When Lora loaded, text shrank 12.5%, causing ~1.0 CLS on 72 feature cards.

**Correct fix:** Lora cap height is ~104% of Georgia's. Georgia needs to render ~4.5% **smaller**, not larger.

```css
size-adjust: 95.5%;
ascent-override: 95%;
descent-override: 25%;
```

Georgia now renders ~4.5% smaller to closely match Lora metrics → minimal shift when font swaps.
