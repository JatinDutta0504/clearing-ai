## Issues Found

### 1. Font 404s (CRITICAL — 2 console errors)
- **Cause:** Lora woff2 URLs in inline `@font-face` declarations returning 404
  - `https://fonts.gstatic.com/s/lora/v37/0QIvMX1D_JOuMwr7I_FMl_E.woff2` → 404
  - `https://fonts.gstatic.com/s/lora/v37/0QIvMW5D_JOuMwr7J_FMl_E.woff2` → 404
  - `https://fonts.gstatic.com/s/lora/v37/0QIvMS1D_JOuMwr7I_FMl_E.woff2` → 404
- **Fix applied:**
  1. Changed all 3 `@font-face` declarations from woff2 → TTF format
  2. Updated to current Google Fonts API TTF URLs (verified 200 OK):
     - `0QI6MX1D_JOuGQbT0gvTJPa787weuyJG.ttf` (Lora 400 normal) ✅
     - `0QI8MX1D_JOuMw_hLdO6T2wV9KnW-MoFkqg.ttf` (Lora 400 italic) ✅
     - `0QI6MX1D_JOuGQbT0gvTJPa787zAvCJG.ttf` (Lora 600 normal) ✅
  3. Changed `font-display: swap` → `font-display: optional` (eliminates font-swap CLS)
  4. Removed stale `<link rel="preload">` tags for woff2 files (were causing 404s)
  5. Removed Google Fonts preconnect links (not needed — no Google Fonts stylesheet used)

### 2. Quiz Button Dark Mode Contrast (Accessibility FAIL)
- **Issue:** `.quiz-start-btn` dark mode text color `#0f1f0f` on `#1e2b1e` background = 1.16:1 contrast (invisible!)
- **Root cause:** `[data-theme="dark"] .quiz-start-btn` rule existed but lacked `!important` — overridden by `.quiz-start-btn` rule's `!important`
- **Fix applied:** Added `!important` to all dark mode properties:
  ```css
  [data-theme="dark"] .quiz-start-btn {
    background: #1e2b1e !important;
    color: #c8d5cb !important;
    border-color: #1e2b1e !important;
  }
  [data-theme="dark"] .quiz-start-btn:hover {
    background: #0f1f0f !important;
    color: #f5f0e8 !important;
  }
  ```

### 3. Nav CTA Contrast (Accessibility FAIL)
- **Issue:** `.nav-cta` background `#4a8a5a` on white text = 4.13:1 (needs 4.5:1 for small/bold text)
- **Fix applied:** Darkened green `#4a8a5a` → `#3d7a4d` (4.68:1 contrast ratio ✅)

### 4. Aria-Label Mismatches (Accessibility FAIL)

  - `aria-label="For Hacker News Readers"` → visible: "📌 For Hacker News Readers"
- **Fix applied:** Updated all 3 aria-labels to include emoji prefix
