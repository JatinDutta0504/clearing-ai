### Duplicate CSS Async Link Fix (CLS improvement)

Discovered and fixed a genuine bug: **7 pages had duplicate non-render-blocking CSS `<link>` tags** — two identical `<link rel="stylesheet" href="css/style.min.css" media="print" onload="this.media='all'">` lines on the same page. This causes:
- Double network fetch of style.min.css (50KB)
- Potential CLS from late-loading CSS overwriting earlier styles
