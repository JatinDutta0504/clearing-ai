## Tweet 2 (The Definition)
The "Middleman Problem":

You write the prompts.
The AI writes the code.
You clean it up.
You ship it.

Your name is on the commit. But the thinking that used to happen in the code — that's gone.

---

## Tweet 3 (The Three Signals)
How to know if you have it:

1. You can't explain how the code works without the AI layer
2. You spend more time curating than writing
3. Code reviews make you quietly anxious

Not imposter syndrome. Skill atrophy.

## Tweet 5 (Why It's Not Imposter Syndrome)
Imposter syndrome = "I'm not good enough."
Middleman problem = "I used to be good at this and I'm not anymore."

One is cognitive distortion. The other is structural.

---

## Tweet 6 (The Recovery Path)
Two practices that actually work:

1. Explanation Requirement: Explain your code out loud before touching AI. Watch what surfaces.
2. No-AI blocks: One hour per week, start from absolute scratch.

Not for productivity. For craft.
